Editor General Kow-How A driver/device driver is a piece of software That software act as an interface to hardware devices, enabling OS and other computer programs to access hardware functions without needing to know precise details about the hardware being used All I/O devices required device drivers such as keyboards, mice, CD/DVD drives, controllers, printers, graphics cards and ports If the driver is included in an operating system, it may be referred to as a kernel-mode device driver Typically, a driver is a dynamic-link library (DLL) with the .sys file name extension A driver communicates with the device through the computer bus or communications subsystem to which the hardware connects When a calling program invokes a routine in the driver, the driver issues commands to the device (drives it) Once the device sends data back to the driver, the driver may invoke routines in the original calling program Routine is a sequence of program instructions that performs a
